# The Python Software Foundation is an organization devoted to advancing open source technology related to the Python programming language. 
# We support the Python Community through...
#  Grants 
In 2023, the PSF awarded $697,000 USD to 174 groups or individuals in 52 countries around the world. 
#  Infrastructure 
We support and maintain [python.org](https://www.python.org/psf/<https:/python.org>), [The Python Package Index](https://www.python.org/psf/<https:/pypi.org>), [Python Documentation](https://www.python.org/psf/<https:/docs.python.org>), and many other services the Python Community relies on. 
#  PyCon US 
We produce and underwrite the [PyCon US Conference](https://www.python.org/psf/<https:/us.pycon.org>), the largest annual gathering for the Python community. Our sponsorsâ€™ support enabled us to award more than $280,000 USD in financial aid to 374 attendees for PyCon 2023.
